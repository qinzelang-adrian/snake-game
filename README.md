# snake-game

一个使用 Python `pygame` 编写的经典贪吃蛇游戏。

## 功能特点

- 经典贪吃蛇玩法：方向键控制移动，吃食物增长
- 碰撞检测：撞墙或撞到自己身体会结束游戏
- 实时计分显示
- 游戏结束后按空格键重新开始

## 运行方法

需要安装 Python 3.x 以及 `pygame` 库：

```bash
pip install pygame
python 贪吃蛇游戏.py
```

## 操作说明

- 方向键（↑ ↓ ← →）：控制蛇的移动方向
- 空格键：游戏结束后重新开始
